# Fable-UUID

> A simple, flexible UUID generator for Node.js and browsers

- RFC 4122 v4 compliant UUID generation with cryptographic randomness
- Configurable random string mode for custom-length identifiers
- Smooth browser support with WebCrypto API fallbacks
- Lightweight Fable service provider integration
